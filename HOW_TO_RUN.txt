PAMS - Paragon Apartment Management System
How to Run

REQUIREMENTS
- Python 3.10+
- MySQL 8.0 running on localhost
- MySQL root password: 314159Pi.com

Install Python dependencies:
  pip install mysql-connector-python matplotlib

FIRST-TIME SETUP (run once)
1. Open a terminal in this project folder.
2. Run the database setup script:
     python -m backend.database.setup_db

   This will:
   - Create the 'asd_project' database
   - Create all 9 tables
   - Populate mock data (properties, tenants, leases, invoices, payments, etc.)

RUN THE APPLICATION
  python main.py

WHAT EACH ROLE CAN DO
Admin       - Manage all user accounts, view/update apartments,
              track all leases (including expiry alerts), and view
              occupancy, financial, and maintenance reports for their city.

Manager     - View multi-city occupancy charts, financial reports,
              maintenance cost reports, and expand business to new cities
              by registering new properties.

Front Desk  - Register new tenants (all required fields including NI number,
              references, occupation, lease period), view all tenants,
              log maintenance requests, and log tenant complaints.

Finance     - View financial overview with month/year filter, browse
              all invoices and payments, record rent payments (generates
              a receipt number), and view overdue invoice alerts.

Maintenance - View open/in-progress requests (priority ordered), assign
              staff to requests, mark requests as resolved with time taken
              and cost recorded, and view full history.

Tenant      - View personal details and active lease, view payment history
              with month/year filter, submit maintenance requests,
              submit complaints, and request early lease termination
              (1 month notice, 5% of monthly rent penalty).

DATABASE
The database name is: asd_project
To export a fresh dump after setup:
  mysqldump -u root -p asd_project > Dump_PAMS.sql
